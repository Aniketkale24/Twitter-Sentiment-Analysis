# Sentiment Analysis on Twitter Data

This project classifies tweets into positive, negative, and neutral sentiments using NLP techniques.

## Technologies Used
- Python
- Scikit-learn
- TF-IDF Vectorization
- LinearSVC

## Features
- Text preprocessing
- Multi-class classification
- Real-time sentiment prediction

## How to Run
1. Install dependencies
2. Run: python main.py
3. Enter tweet text to get sentiment

## Output
The model shows accuracy and classification report along with live predictions.
